# MsnhnetSharp
---
**C# wrapper for msnhnet.**
### Msnhnet
https://github.com/msnh2012/Msnhnet
### How to USE
- 1 Build msnhnet with cmake (Release mode).
- 2 Copy msnhnet.dll to MsnhnetSharp dir.(Debug or Relase)
- 3 Open this project with Visual Stuido.
